相对路径依赖于当前工作目录 (CWD)
代码中的 Usd.Stage.Open("assets/Female_T_Shirt.usd") 使用相对路径，这意味着路径是相对于 Python 脚本运行时的当前工作目录 (CWD) 解析的。CWD 由运行命令的位置决定，而不是脚本文件的位置

修复建议
使用基于脚本目录的绝对路径，避免依赖 CWD：
```
import os
# ... existing code ...
usd_state = Usd.Stage.Open(os.path.join(os.path.dirname(__file__), "assets/Female_T_Shirt.usd"))
# ... existing code ...
```
os.path.dirname(__file__) 获取脚本所在目录（tutorial），然后拼接 assets/Female_T_Shirt.usd。
确保 assets 文件夹在 tutorial 下，或调整路径指向 newton_physic_sim/assets（例如 os.path.join(os.path.dirname(__file__), "../../assets/Female_T_Shirt.usd")）。
